# Sales-data-analysis-
This Power BI dashboard analyzes sales, profit, quantity, and promotions using interactive visuals. It includes top/bottom product insights, sales trends, period comparison, city-wise sales mapping, and a detailed order table for deeper investigation.
Key Features

1.Executive Overview: Total orders, net sales, profit, discounts, and city-wise sales map.

2.Product Insights: Top 5 & Bottom 5 products by sales, profit, and quantity sold.

3.Sales Trends: Multi-year time-series showing demand patterns and peak periods.

4.Promotion Analysis: Average discount by promotion category and campaign impact.

5.Period Comparison: Compare sales, profit, and quantity between two date ranges.

6.Detail Table: Fully filterable dataset with customer, product, discount, net sales, and profit.
